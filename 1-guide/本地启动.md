# 本地启动指南

## 环境要求

- Node.js >= 16.0.0
- Yarn
- Git

## 安装步骤

### 1. 安装依赖

```bash
yarn install
```

### 2. 启动开发服务器

```bash
yarn start
```

启动后访问 http://localhost:3000

## 其他命令

| 命令 | 说明 |
|------|------|
| `yarn build` | 生产环境构建 |
| `yarn test` | 运行测试 |
| `yarn lint` | 代码检查 |
| `yarn build --stats` | 构建并生成包分析文件 |

## 常见问题

### 主题构建失败

项目启动时会先执行 `yarn buildThemes` 编译 Less 主题文件，确保已安装 `less` 和 `less-plugin-clean-css`。

### 端口占用

默认端口 3000，如被占用会提示切换到其他端口。

### Node 版本问题

项目要求 Node 16.x，建议使用 nvm 管理 Node 版本：

```bash
nvm install 16
nvm use 16
```
